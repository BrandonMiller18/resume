A bit about me.

My name is Brandon Miller. I am born and raised in St. Louis, MO. I played baseball at Lindewnwood University for a few years before blowing my arm out and undergoing Tommy John surgery (twice). I grew up on sports. I bleed blue, as they say, and by my count I have only missed watching four Blues games in the past 3 years. If I had the money, I would buy the building pictured here and live there full time.

About this website.

I built this website as a passion project to learn and demonstrate my skills. The backend of this site is built using the Python webframework called Flask. There is an admin section of this site (that you can't see) which requires a login and allows me to manage various tables in the database. You can leave a reference for me, which I must first approve in the admin, that is stored in a SQL database then displayed on the front end. The site is hosted on Heroku servers.

Updated periodically. Take a look.

As with most websites, this is a continuous work in progress. There are likely some mistakes as I have been going at this solo. I plan to continually work to improve and add features to the site as time goes on. Pictured here is my home office setup. This whole thing was made sitting at that gorgeous spot. Please check back periodically to see what crazy stuff I have come up with!
