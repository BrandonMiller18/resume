NHL Companion

The NHL Companion is a site I built which allows you to follow your favorite NHL team's game. The app plays your team's goal horn when your team scores.

It uses the NHL's public API to retrieve data and is built using Flask as the framework and SocketIO to manage the communication from server to client. The code is available on my GitHub.

Sofia & Grace

Sofia & Grace is a Magento 2 ecommerce store that I led the development of. The site launched in Q1 2021. The site and brand is focused on destimgatizing urinary incontinence while empowering women to live fuller, healthier lives.
